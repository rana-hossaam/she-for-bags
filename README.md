# She for Bags - Premium E-Commerce Website

A modern, premium e-commerce website for a handmade bags brand built with React, TypeScript, and Tailwind CSS.

## Features

### Customer Features
- Elegant luxury fashion design with beige, black, and gold color scheme
- Responsive design for mobile, tablet, and desktop
- Beautiful hero section with promotional banner carousel
- Featured products section
- Product categories with visual grid
- Product listing page with advanced filters (category, price, color, sort)
- Product details page with image gallery, color/size selection
- Shopping cart with quantity management
- Wishlist functionality
- User authentication (Login/Register)
- Customer profile page with addresses and settings
- Checkout page with multi-step flow
- Order history page
- Contact Us page with form
- About Us page with brand story
- Newsletter subscription section
- Product search feature

### Admin Dashboard
- Analytics dashboard with real-time sales overview
- Total revenue statistics with daily/weekly/monthly views
- Order management with status tracking
- Customer management
- Product management (Add/Edit/Delete)
- Sales statistics charts (Line, Bar, Pie, Area charts)
- Order status distribution
- Customer activity insights
- Top selling products tracking
- Admin-only access protection

## Tech Stack
- React 18
- TypeScript
- Tailwind CSS
- React Router DOM
- Recharts (for analytics charts)
- Lucide React (icons)
- Firebase Authentication (ready for integration)
- Firestore Database (ready for integration)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure
```
src/
  components/       # Reusable UI components
  pages/           # Page components
  admin/           # Admin dashboard components
  context/         # React context (Auth, Cart, Wishlist)
  types/           # TypeScript types
  utils/           # Utility functions
  data/            # Sample data
```
